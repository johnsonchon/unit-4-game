Shroom Collector!

Click on a shroom to add a specific amount of points to add up to "Shrooms needed".

You win the game by matching "Shrooms eaten" to "Shrooms needed". Mario dies if "Shrooms eaten" goes above "Shroom needed".

The value of each shroom is hidden from you until you click on it.

Each time when the game starts, the game will change the values of each shroom.
